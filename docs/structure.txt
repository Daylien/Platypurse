The project is organised in multiple folders, this document will describe those folders

.docker
Docker related files, configuration and building

- app
The "app" folder bootstraps the MVC pattern

- config
Configuration file for the application

- db
Holds sql files and migrations for the database

- docs
Documentation and information about the project

- lib
Base library's for the application the components for the MVC pattern lives in there


- public
Frontend related code, for example is css or js stuff

- scripts
Building utilities and other small pieces that perform some magic


- test
Unit testing files for the application, library have there own test folder
